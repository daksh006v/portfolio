## ✨ Features

* **Modern UI/UX:** Clean, minimalistic, and intuitive interface with carefully crafted animations.
* **Interactive Elements:** Features 3D integrations (Three.js), dynamic micro-interactions, and visual fidelity using Framer Motion.
* **SEO Optimized:** Semantic HTML architecture, dynamic Next.js Metadata API integration, accessible `aria` labels, and lazy-loading for maximum PageSpeed Core Web Vitals.
* **Responsive Design:** Seamlessly scales across mobile, tablet, and desktop viewports.
* **Dynamic Analytics:** Real-time integration of GitHub contributions using `react-github-calendar` and rich stat visualizations alongside LeetCode metrics.
* **Integrated Multimedia:** Engaging multimedia sections using `use-sound` and `react-player`.

## 🚀 Getting Started

Follow these steps to run the portfolio locally.

### Prerequisites

* Node.js (v20+ recommended)
* npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/daksh006v/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or yarn / pnpm install
   ```

3. **Set up environment variables**
   Rename `.env.example` to `.env.local` and fill in the required values.
   ```bash
   cp .env.example .env.local
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
